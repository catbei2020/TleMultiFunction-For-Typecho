### Typecho-TleMultiFunction For Typecho V1.0.6多功能插件
---

Typecho多功能插件

#### 使用方法：
第一步：下载本插件，放在 `usr/plugins/` 目录中；
第二步：激活插件；
第三步：填写配置；
第四步：完成。
突然忘记第五步，如果你的网站根目录下没有配置伪静态的话，可将此插件跟目录下的.htaccess复制到网站根目录（如果子目录的话需要做修改）即可。（如果不会可与作者联系）

#### 使用注意：
此插件使用php5.6编写，如果遇到语法报错，就适当降低php版本即可。
